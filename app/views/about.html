Implementation Details

Github Link

  • Done using AngularJS/JavaScript, HTML5, CSS3, and Yeoman
  • Can search through campaigns by title or tagline or filter by category
  • Clicking campaign will link to corresponding Indiegogo campaign page
  • Progress bar uses Indiegogo color and shows percentage of funds collected compared to the goal
  • Shows how many days left until funding campaign is over
  • Most of the relevant code for this challenge can be found in /app/scripts and /app/views
  • 19 Karma/Jasmine unit tests can be found in /test/spec/controllers/campaigns.js
